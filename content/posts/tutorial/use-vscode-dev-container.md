### 介绍

`Visual Studio Code Remote - Containers`功能能够让你把Docker 容器作为一个开发环境，你能在容器内打开任何的
文件夹和仓库，并且使用VS Code的所有特性，要使用该特性我们需要在项目根目录创建一个``devcontainer.json`VS Code 如何访问运行时的开发容器。此容器可用于运行应用程序或分离使用代码库所需的工具、库或运行时。
